## 文档流与浮动
1. 文档流
   页面只有一层
2. 浮动
   1. 开启浮动后，页面分为多层
   2. 浮动标签不会撑起父标签，父标签的高度为0，除非父标签设定了高度
   3. 浮动标签的非浮动的兄弟标签会填充父标签，并终结浮动，
   4. 带有 "clear:both" "clear:right" "clear:left" 的兄弟标签会终结浮动，并用合适的高度撑起父标签
   5. 文字会被尽量显示
